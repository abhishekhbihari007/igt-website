### **Prerequisites**
- ⚡ Node.js 16+ 
- 📦 npm or yarn
- �� Git

### **Installation**
\`\`\`bash
# 🚀 Clone the repository
git clone https://github.com/abhishekhbihari007/igt-website.git

# 📁 Navigate to project directory
